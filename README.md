# 操作说明
### 选择：
#### w 或↑ 向上
#### s 或↓ 向下
#### 空格 开始游戏

### player1:

#### w 向上
#### s 向下
#### a 向左
#### d 向右
#### 空格 攻击

### player2:

#### ↑ 向上
#### ↓ 向下
#### ← 向左
#### → 向右
#### 回车 攻击
